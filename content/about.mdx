# About Alfredo

I’m Alfredo Navas, a developer building a small personal website that stays fast, readable, and easy to query.

This site is designed as a set of compact knowledge nodes, with each page keeping a focused scope and a clear purpose.

I like minimal interfaces, semantic structure, and content that can be reused cleanly by both humans and tools.
